CSS Syntax

Below is a breakdown of a CSS block of code.

h3 {
background-color: blue;
color: white;
}

  • h3: this is called selector, selectors can be html elements, classes, ids and attributes.
    Selectors are followed by curly braces {} with the styling code written inside.

  • background-color: and color: are called properties.

  • blue: and white: are the values to the respective properties.

  • Property and value pairs are called declarations and after each declaration we place a semi-colon ( ; ).
