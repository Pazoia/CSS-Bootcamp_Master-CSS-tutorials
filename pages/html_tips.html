HTLM Tips

Below are some tips to make writing HTML quicker

  • Creating an html file:
    to start an html file type ! followed by the tab or enter key.

  • Creating one element in html:
    to create an element in your html file type the element name for example
    div followed by the tab or enter key.
    This will create the opening and closing tags.

  • Creating multiple elements in html:
    to create multiple elements in your html file type the element name * the number of elements required for example
    div*10 followed by the tab or enter key.
    This will create 10 div elements with opening and closing tags.

  • Creating an element with a child element in html:
    to create an element with a child element inside it in your html file
    type the parent element name + > + child element name for example
    div>h1 followed by the tab or enter key.
    This will create one div element with an h1 element in it, both with opening and closing tags.

  • Creating an element with an attribute in html:
    to create an element with an attribute in your html file
    type the element name + . or # + desired attribute name for example
    h1.heading or h1#heading followed by the tab or enter key.
    The h1.heading will create an h1 element with the class attribute with opening and closing tags.
    The h1#heading will create an h1 element with the id attribute with opening and closing tags.
