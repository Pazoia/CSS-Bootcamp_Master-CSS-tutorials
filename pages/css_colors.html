CSS Colors

Below are some of the ways to choose and apply colors in CSS

  • This is not the most popular way but you can simply type the color name, for ex:
    color: red;
    color: green;
    color: blue;

  • Choose a color using its rgb values.
    rgb values range from 0 to 255, below is the way to choose a color this way:

    rgb(255, 0, 0) = Red
    rgb(0, 255, 0) = Green
    rgb(0, 0, 255) = Blue

    CSS 3 added rgba, which is the option to select the opacity of a color.
    rgba( 255, 0, 0, 0.5) = Red

  • Choose a color using its Hexadecimal values.
    Hexadecimal values range from 0 to f, below is the way to choose a color this way:

    #ff 00 00 = Red
    #00 ff 00 = Green
    #00 00 ff = Blue
