CSS Combinators

A combinator defines the relationship between selectors.

There are 4 types of CSS Combinators

  • Descendant(parent followed by element) - Matches all dependant elements of specified element.
    section p {}
    This would apply the styling to all p elements under section.

  • Child( > ) - The child combinator selects all immediate children of specified element.

    section>
    p> Paragraph 1 /p>
    ...div>
    ...p> Nested Paragraph 2 /p>
    .../div>
    p> Paragraph 3 /p>
    /section>

    section > p {}
    In this case the styling would be applied to Paragraphs 1 and 3,
    Nested Paragraph 2 is nested in a div, therefore not an immediate child of section.

  • Adjacent sibling( + ) - This selects the following immediate sibling of a specified element.

    section>
    h1>Heading /h1>
    p> Paragraph 1 /p>
    ...div>
    ...p> Nested Paragraph 2 /p>
    .../div>
    p> Paragraph 3 /p>
    /section>

    h1 + p {}
    In this case the styling would be applied to Paragraph 1 only,
    it's the h1 immediate following sibling.

  • General sibling( ~ ) - This selects all siblings placed after a specified element.

    section>
    h1>Heading /h1>
    p> Paragraph 1 /p>
    ...div>
    ...p> Nested Paragraph 2 /p>
    .../div>
    p> Paragraph 3 /p>
    /section>

    h1 ~ p {}
    In this case the styling would be applied to Paragraphs 1 and 3,
    Nested Paragraph 2 is nested in a div, therefore not a sibling of h1.
