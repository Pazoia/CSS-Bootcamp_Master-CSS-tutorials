CSS Selectors

In this lesson we explore how to use CSS selectors.

  • Precedence in order of importance:
    element with parent element has higher precedence than just the element.
    ul li {} will override li {}

    class selection has higher precedence than element selection.
    .item {} will override ul {}

    id selection has the highest precedence than all the others.
    #item {} will override .beer {}
    #item {} will override h1 {}
    #item {} will override ul li {}

  • More than one selector can be used in one line,
    for example if both h1 and h2 require the same CSS styling
    the block of code would look like:

    h1, h2 {
    // styling code inside
    }

  • To access an element using it's class to select it,
    type .class followed by curly braces. This will affect all
    elements with this class.

    h1 class="wine"
    li class="wine"

    .wine {
    // styling code inside
    }

    In this case both h1 and li would have the same styling.

  • To access an element using it's id to select it,
    type #id followed by curly braces. This will affect all
    elements with this id.

    h1 id="beer"
    li id="beer"

    #beer {
    // styling code inside
    }

    Although the id would also use the same styling for both
    scenarios like in the case of the class, it is recomended that
    id is to be used for an individual element styling, whilst class
    is to be used to style various elements the same way.
