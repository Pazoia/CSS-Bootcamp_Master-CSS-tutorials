Styling Methods

There are three styling methods

  • Inline Method - The CSS style is applied directly to the HTML element.
    In the case of multiples of the same element with the same styling
    there would be plenty of unneccessary repeated code
    (Not best practice)

    • Inline Styling

  • Internal Method - The CSS code is written inside style tags, inside the head tags of the HMTL code.
    Although this is better than the inline method it is still only applicable withing the same file,
    in the case of a big project, this code would have to be repeated in various files.
    (Not best practice)

    • Internal Styling

    Internal Styling

  • External Method - Using the external method requires two things:
    First create a .css file where the CSS styling code will be written.
    Secondly, link the .css file with the file where the styling will be applied,
    using the link tags at the head of the HTML code.
    Advantages of using this method are that the code will be easier to locate in case of a change,
    and one block of code can work for various documents avoiding repetition of code.

    Preferred Method

    • External Styling
